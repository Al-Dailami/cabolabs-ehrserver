Re: HttpBuilder: XML POST example

791 posts
Well, to be honest I've had a hard time finding any public APIs that
use POSTs that accept content-types other than form-url-encoded.  So I
don't even have a working unit test for it.  But you should be able to
set the 'body' property as if it was a StreamingMarkupBuilder.bind
definition:

http.request( POST, XML ) {
  body = {
    qdbapi {
      username "PTBarnum"
      password "..."
    }
  }
}

Found this to work, assuming that you have your xml as a string

http.request( POST) {
        delegate.contentType = "text/plain"
        delegate.headers['Content-Type'] = "text/xml"
        delegate.body = <your xml string>
       

         .....
